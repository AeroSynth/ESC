# ESC
Code in Python to run the voice demo for Eugene Science Center
GUI developed using Page.  Main program are:
* brad_mpl.py -- generated GUI by Page.  Do not modify.
* brad_mpl_helper -- code to control program in response to widgets
* Audio wavefiles are located in 6 sub-directories

to run, use "python brad_mpl.py" on the command line (or "python3 brad_mpl.py")
